## Installation

1. Install the `@radix-ui/react-toggle` component from radix-ui:

```bash
npm install @radix-ui/react-toggle
```

2. Copy and paste the following code into your project.

<ComponentSource src="/components/ui/toggle.tsx" />

<Callout>
  This is the `<Toggle />` primitive. You can place it in a file at
  `components/ui/toggle.tsx`.
</Callout>

## Usage

```tsx
import { Toggle } from "@/components/ui/toggle"
```

```tsx
<Toggle>Toggle</Toggle>
```
